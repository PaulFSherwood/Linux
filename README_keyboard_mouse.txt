# Keys will not auto repeat 
# i.e. you type something into xterm hold backspace to 
# -- erase it and the key is only hit once.  doesn't 
# -- remove all the text

# to reset use xset (Valuse 170 and 30 are good start point)
# https://wiki.archlinux.org/index.php/Xorg#Key_repeat_delay_and_rate
xset r rate 170 30
